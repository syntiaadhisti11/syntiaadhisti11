## 🌸 Tentang Aku

Aku mahasiswa Teknik Informatika di **Universitas Islam Riau (UIR)**, suka membangun aplikasi berbasis **AI** dan web/mobile dengan tampilan yang nggak cuma jalan tapi juga enak dilihat. Beberapa hal yang aku suka eksplor:

- 🤖 Mengintegrasikan LLM lokal (Ollama) ke dalam aplikasi nyata — dari deteksi hoax berita, analisis kulit, sampai keamanan siber
- 🎨 Desain UI yang punya karakter — glassmorphism, tema futuristik, sampai pastel yang lembut
- 📱 Membangun aplikasi mobile dengan React Native
- 🏆 Ikut kompetisi seperti Satria Data dan program beasiswa Pertamina Foundation
